# Cloud Conformity Rule List

Extracts information about Cloud One Conformity rules.

## Usage

To use the script, specify the following required environment variables:

  * `CC_API_KEY`
  * `CC_REGION`

Then run the following command:

```
python3 get_rules.py
```
